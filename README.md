# Happy-gyc Blog

这是 [NGC2244 / Happy-gyc](https://gyc-blog.cc.cd) 的个人博客源码，基于 [Firefly](https://github.com/CuteLeaf/Firefly) 进行二次定制。仓库主要用于维护个人站点的配置、内容和少量功能调整。

## 致谢

感谢 [CuteLeaf](https://github.com/CuteLeaf) 与所有 [Firefly 贡献者](https://github.com/CuteLeaf/Firefly/graphs/contributors) 的开发和维护。Firefly 提供了 Astro + Svelte 的站点结构、响应式布局、壁纸与侧边栏系统、文章页面、搜索、多语言支持等基础能力，本项目的个性化修改都建立在这些工作之上。

Firefly 基于 [saicaca/fuwari](https://github.com/saicaca/fuwari) 开发，也感谢 [saicaca](https://github.com/saicaca) 及 Fuwari 的贡献者。原项目的开放共享让这个个人博客省去了大量从零搭建和维护的工作。

## 个性化修改

- 将站点名称、头像、导航、个人资料、公告和页面内容调整为 NGC2244 / Happy-gyc 的个人博客配置。
- 制作并接入适配亮色、暗色主题及桌面、移动设备的 NGC2244 壁纸，默认使用全屏透明模式，同时调整主题色、卡片、侧边栏和页面布局。
- 接入霞鹜文楷本地字体与构建时字体子集化，并调整樱花、环境光效、壁纸轮播和页面过渡效果。
- 按个人使用需求配置 Giscus 评论、音乐播放器、留言页和内容页面，并补充 Pages CMS 内容编辑与 Cloudflare 部署配置。

后续修改仍以个人使用需求为主。Firefly 的完整功能、配置方法和使用文档请查看 [Firefly 官方仓库](https://github.com/CuteLeaf/Firefly) 与 [使用文档](https://docs-firefly.cuteleaf.cn/)。
